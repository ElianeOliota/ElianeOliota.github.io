# Documentação
## Descrição:
Essa funcionalidade tem como objetivo calcular os sintomas do usuário e oferecer um resultado de acordo com a gravidade dos sintomas. Sendo de baixa urgência, será recomendado monitorar a saúde, utilizar máscaras e álcool em gel, e manter distanciamento social. Sendo moderado, será recomendado a busca de um posto de saúde mais próximo. Sendo grave, será recomendado a busca de hospitalização no hospital mais próximo. A busca dos hospitais e postos de saúde será feita pela API do Google Maps

## Detalhes técnicos:
Essa funcionalidade poderá ser implementada com HTML, CSS e JavaScript. Com consulta a API do Google Maps.
A busca pelos postos de saúde ou hospitais será feita utilizando a geolocalização do usuário.
